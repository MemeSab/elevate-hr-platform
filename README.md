# 🧠 Intelligent OD – Premium HR Consultancy & Learning Platform

## Overview
A modern, high-authority digital presence for **Intelligent OD**, blending professional organisational development consultancy with an elite video-based learning experience.

This project explores how a consultancy brand can leverage SaaS-inspired aesthetics and interactive local-first learning tools to bridge the gap between traditional advisory and modern digital education.

---

## The Problem
Most HR consultancy websites rely on generic layouts and text-heavy descriptions, which fail to convey authority or engage modern leaders.

In the Organisational Development (OD) space, there is also a disconnect between high-level theory and actionable learning for HR professionals.

This can lead to:
- Low brand engagement
- Friction in service discovery
- Inefficient knowledge transfer for teams

---

## The Solution
**Intelligent OD** introduces a minimalist, visual-first platform that combines strategic service deep-dives (Bento Grid) with a structured Four-Step Framework.

It allows users to:
- Understand complex OD frameworks at a glance
- Preview elite video courses via an interactive paywall system
- Access "Latest Insights" from Kash Haroon’s professional leadership

---

## ✨ Features

- **Premium Glassmorphic UI**: Modern SaaS-inspired aesthetics with frosted-glass navigation and dynamic animations.
- **Strategic Bento Grids**: Highly responsive service and insight layouts that scale across all devices.
- **Interactive Framework**: Visualised "Four-Step Framework" for organisational transformation.
- **Video Learning Platform**: Custom video player with 15-second trial previews and login-to-continue logic.
- **Global Navigation System**: Unified, mobile-optimised overlay menu for narrow portrait viewports (400px).
- **SEO Optimized**: Semantic HTML5 structure with optimized meta tags and performance-focused asset handling.

---

## 🔧 Tech Stack

- **Frontend**: HTML5, Vanilla CSS3 (Custom Design System)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Backend**: Firebase (Integration ready for Auth/Dashboard)
- **Animations**: GSAP & CSS Keyframes
- **Deployment**: Git-ready for GitHub Pages / Vercel

---

## 🧠 Key Decisions

- **Visual over Text**: Chose a "Bento-style" layout to improve scanning of complex service offerings.
- **Mobile-First Navigation**: Replaced traditional menus with a full-screen glass overlay to ensure clarity on narrow screens (400px).
- **Minimalist Palette**: Used a curated navy and slate dark mode to establish immediate trust and authority.
